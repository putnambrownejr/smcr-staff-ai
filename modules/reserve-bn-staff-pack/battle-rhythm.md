# Reserve Battalion Battle Rhythm (UNCLASSIFIED)

## Drill Weekend Standard Timeline

### Saturday

| Time  | Event                                | Lead   | Notes                                       |
|-------|--------------------------------------|--------|---------------------------------------------|
| 0700  | I&I staff opens / range setup        | I&I OpsChief | Verify range is hot, ammo drawn if applicable |
| 0745  | Officer/SNCO call                    | XO     | Hot items, training schedule changes, admin fires |
| 0800  | Formation / accountability / safety brief | 1stSgt | Muster report to S-1 within 30 min         |
| 0830  | Commander's time                     | CO     | 15 min max — guidance, recognition, message |
| 0845  | Training block 1                     | S-3    | Per published training schedule              |
| 1130  | Noon meal                            | S-4    | Headcount to messhall/caterer by D-7        |
| 1230  | Training block 2                     | S-3    |                                              |
| 1600  | AAR / training debrief               | Trng Off | Capture sustains/improves before release    |
| 1630  | End-of-day formation                 | 1stSgt | Liberty brief, Sunday report time            |

### Sunday

| Time  | Event                                | Lead   | Notes                                       |
|-------|--------------------------------------|--------|---------------------------------------------|
| 0800  | Formation / accountability           | 1stSgt |                                              |
| 0815  | Training block 3                     | S-3    |                                              |
| 1130  | Noon meal                            | S-4    |                                              |
| 1230  | Admin / make-up training / PME       | S-1/S-3 | Catch Marines who missed Saturday events    |
| 1400  | Staff sync / due-out review          | XO     | Confirm all drill-weekend deliverables met  |
| 1500  | Gear turn-in / police call           | S-4    |                                              |
| 1530  | End-of-drill formation               | 1stSgt | Next drill date, travel reminders           |

## Inter-Drill Staff Rhythm (Monthly)

| When        | What                                     | Who    |
|-------------|------------------------------------------|--------|
| Drill + 5   | Post-drill admin complete (Drill Mgr, DTS, MROWS) | S-1    |
| Drill + 7   | AAR written and filed                    | S-3    |
| D-30        | Training schedule to CO for approval     | S-3    |
| D-21        | MROWS orders submitted (if AT/ADT)      | S-1    |
| D-14        | Training materials / lesson plans due    | Trng NCOs |
| D-14        | Range/facility reservations confirmed    | S-3    |
| D-7         | Logistics coordination complete (ammo, chow, billeting) | S-4 |
| D-7         | Admin readiness check (pay, orders, travel) | S-1 |
| D-3         | Safety brief / ORM worksheet published   | S-3    |
| D-1         | Pre-execution checks complete            | XO     |
