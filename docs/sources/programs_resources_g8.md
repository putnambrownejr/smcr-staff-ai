# Programs, Resources, And G-8 Grounding

This note captures official public grounding for G-8, comptroller, and resource-tradeoff support. It is a reference note, not a copied order.

## Official Sources

- MCO 7300.21B
  - Official URL: https://www.marines.mil/News/Publications/MCPEL/Electronic-Library-Display/Article/900441/mco-730021b/
  - Use for: Marine Corps financial-management principles, controls, operations, and stewardship.
  - Notes: core baseline for resource-estimate language and fiscal-control framing.

- MARFORRES G-8 Comptroller Resources
  - Official URL: https://www.marforres.marines.mil/Staff-Sections/General-Staff/G-8-Comptroller/Resources/
  - Use for: reserve-specific resourcing posture, internal control, audit support, and limited-resource decision support.
  - Notes: especially relevant for SMCR framing and reserve resourcing realism.

- Marine Corps Logistics Command G-8 Comptroller
  - Official URL: https://www.logcom.marines.mil/Staff-Offices/G-8-Comptroller/
  - Use for: PPBE, budgeting, execution oversight, audit readiness, and financial stewardship.
  - Notes: useful public model for G-8 mission framing beyond a single installation.

## Expected Agent Usage

- resource estimate
- funding-risk note
- priority tradeoff brief
- resourcing decision point

## Safety / Freshness Notes

- Local comptroller guidance, current funding lines, and command authorities remain controlling.
- Do not treat this note as authorization to obligate funds or represent current budget status without local review.
