# Source Reference Library

This directory stores lightweight Markdown references for official public sources the project expects to use later for ingestion, retrieval, citations, and agent grounding.

These files are not copies of orders, manuals, PDFs, or article text. They are quick-reference notes with:

- source title
- source family/category
- official public URL
- expected agent usage
- ingestion notes
- safety or freshness concerns

When real ingestion is added, fetch from the official URL, calculate source hashes, preserve publication metadata, and avoid committing large source text or copyrighted publication bodies.
