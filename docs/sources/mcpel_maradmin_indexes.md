# MCPEL And MARADMIN Indexes

## Primary Sources

- Title: Marine Corps Publications Electronic Library (MCPEL)
- Type: Publication index
- Category: Source index
- Official URL: https://www.marines.mil/News/Publications/MCPEL/
- Classification label: UNCLASSIFIED public index
- CUI flag: false for public entries; MCPEL notes that CUI publications require TDM-PUBS.

- Title: MARADMIN Messages
- Type: Message index
- Category: Administrative messages
- Official URL: https://www.marines.mil/News/Messages/MARADMINS/
- Classification label: UNCLASSIFIED public index
- CUI flag: false

## Expected Uses

- MARADMIN monitor
- MCPEL ingestion
- Doctrine freshness checks
- Source status verification

## Notes

Do not scrape CUI-only publications or authenticated TDM-PUBS content in this public prototype. Tests should use local fixtures and should not require live network access.
