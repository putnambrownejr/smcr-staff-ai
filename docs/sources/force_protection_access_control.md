# Force Protection And Access Control Grounding

This note captures official public grounding for provost, access-control, and force-protection planning support. It is a reference note, not a copied order.

## Official Sources

- MCO 5530.14A Marine Corps Physical Security Program Manual
  - Official URL: https://www.marines.mil/Portals/1/Publications/MCO%205530_14A.pdf
  - Use for: physical-security program framing, responsibilities, and baseline control concepts.
  - Notes: use for annex logic, not as a substitute for local site procedures.

- Release Of Marine Corps Order 5530.13 Marine Corps Site Perimeter Access Control
  - Official URL: https://www.marines.mil/News/Messages/Messages-Display/Article/2622028/release-of-marine-corps-order-553013-marine-corps-site-perimeter-access-control/
  - Use for: perimeter and site access-control posture.
  - Notes: good public anchor when the user is shaping visitor/access assumptions.

- Headquarters Marine Corps Physical Security
  - Official URL: https://www.ar.marines.mil/Branches/ARS-Security-Programs-and-Information-Management/Physical-Security/
  - Use for: official public entry point to Marine Corps physical-security program context.
  - Notes: use when a user needs the current public-facing program home.

- Marine Corps Installation Access Control Policy
  - Official URL: https://www.marines.mil/News/Publications/MCPEL/Electronic-Library-Display/Article/899440/mco-p55302d/
  - Use for: installation access-control framing and visitor-management logic.
  - Notes: pairs well with event-specific access and traffic-control planning.

## Expected Agent Usage

- security annex
- access-control plan
- visitor-control checklist
- traffic and parking control plan
- incident escalation and emergency-action checks

## Safety / Freshness Notes

- Local installation rules and emergency procedures may differ from generic Marine Corps references. Verify locally before execution.
- Do not use this note as authority for detention, search, or current security operations. Human review and local command/legal review still apply.
