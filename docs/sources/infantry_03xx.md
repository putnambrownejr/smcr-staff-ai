## Safety And Usage Notes

Use these references for doctrine-grounded education, training checklists, and fictional/training scenario planning. Do not use the system to process real operational plans, current unit movements, sensitive locations, force protection details, or classified/CUI information.

When connected to RAG later, responses should cite publication title, official MCPEL URL, retrieval date, version/status, and section/page metadata when available.
