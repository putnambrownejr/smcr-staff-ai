## Safety And Usage Notes

Use these references for doctrine-grounded education, training checklists, and fictional/training scenario planning. Do not process real deployment plans, sensitive movement details, current unit logistics posture, exact routes, force protection details, classified information, CUI, private vendor-sensitive information, or protected supply/accountability records.

When connected to RAG later, responses should cite publication title, official MCPEL URL, retrieval date, version/status, and section/page metadata when available.
