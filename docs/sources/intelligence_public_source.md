agent sound academic. The point is to make it careful, selective, and useful.

## Notes

These sources should push the S-2 lane toward:

- one assessment, one key caveat, one information gap
- clear separation of facts, claims, assumptions, and unknowns
- explicit confidence language
- refusal to let noisy sources masquerade as command-grade understanding

Do not overfit this lane to current-events commentary. The main job is decision support and disciplined caveating.
