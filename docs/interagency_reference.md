# Interagency & Joint Operations Reference

**UNCLASSIFIED — Advisory reference only. Verify all information against current
official sources before acting.**

This reference covers practical coordination mechanics for working with external
organizations: other US military branches, allied nations, US government agencies,
and NGOs/international organizations. It answers "who do I contact, what structure
exists, what agreements govern this, and what do I need to set up."

**This is NOT the G-9 / Civil Affairs lane.** G-9 handles the civil estimate,
ASCOPE/PMESII, community engagement, and Annex G. This reference covers the
organizational plumbing — command relationships, liaison, comms interoperability,
agreements, and coordination mechanisms that make joint/interagency/multinational
operations work.
