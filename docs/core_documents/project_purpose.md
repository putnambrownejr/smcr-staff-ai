# Project Purpose

`smcr-staff-ai` is a personal SMCR staff copilot for awareness, readiness, and effective staff action.

It exists to help Marine Corps Reservists stay aware, stay sharp, and stay ahead of the friction of reserve service by turning scattered obligations, staff problems, and training demands into organized, advisory, usable outputs.

## Three Pillars

### 1. Awareness

The project should help the user maintain awareness of:

- drill rhythm
- admin suspense
- readiness gaps
- personal document gaps
- source changes and MARADMIN relevance
- opportunities, PME, and career watch items

Typical features in this pillar:

- Chief brief
- reminder plans
- session handoffs
- local document organization
- source verification and update tracking

### 2. Readiness And Edge

The project should help the user become a sharper planner, staff officer, and Marine.

Typical features in this pillar:

- staff package generation
- FRAGO to CONOP flow
- training scenario design
- MET / METL alignment
- AAR and SITREP support
- current-event-driven case studies
- staff-agent review and command-team critique

### 3. Friction Reduction

The project should reduce the practical burden of reserve life so the user can preserve civilian balance while staying useful to the SMCR.

Typical features in this pillar:

- drill prep
- AT prep
- DTS / GTCC support
- PKI / CAC troubleshooting
- installation / base practical guidance
- continuity between drills

## Product Standard

This project should:

- be SMCR-primary
- stay advisory, not authoritative
- preserve human review
- ground outputs in public sources when possible
- turn context into usable products, actions, and reminders

## In-Scope Heuristic

A feature is probably worth building if it clearly does one or more of these:

1. improves awareness
2. improves readiness or judgment
3. reduces reserve friction

If it does not clearly support one of those three, it is lower priority even if it is interesting.

## Plain-English Version

This repo is trying to help a Marine Reservist keep the reserve burden organized, stay professionally sharp, and do better staff work with less wasted energy.
